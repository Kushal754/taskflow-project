# 🧪 Experimentos de Programación con y sin IA
Este documento recoge una comparativa práctica de resolución de problemas. Analizaré la diferencia entre resolver pequeños retos algorítmicos y tareas del proyecto TaskFlow de forma manual (sin ayuda) frente a hacerlo asistido por IA, midiendo el tiempo invertido, la calidad del código y mi comprensión del problema.

## 🚀 Funcionalidad Extra: Dictado por Voz y Filtros Avanzados

Para llevar el proyecto al siguiente nivel y demostrar la capacidad de expandir código con IA, utilicé el **Composer de Cursor** para implementar dos características avanzadas:
1. **Filtro de Categorías:** Un selector dinámico que se integra con la lógica de filtrado existente.
2. **Dictado por Voz (Web Speech API):** Implementé un botón de micrófono que utiliza la API nativa de reconocimiento de voz del navegador. Al hacer clic, pide permisos de micrófono y transcribe la voz del usuario directamente al campo de texto de la nueva tarea. 

Esto demuestra cómo la IA puede ayudarnos a integrar APIs nativas complejas (como la de voz) en cuestión de minutos, mejorando drásticamente la accesibilidad de la aplicación.
